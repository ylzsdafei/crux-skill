# 蒸馏步骤 2：提炼决策启发式（候选 → 筛选）

## 任务

基于步骤 1 的心智模型 + 重新扫一遍原文，**提取决策启发式清单**。最后筛出：
- 5 条核心启发式（进入对应 Model 的 5 条骨架）
- 3-5 条通用启发式（进入 `universal-heuristics.md` 通用池）

## 启发式格式

- 格式："遇到 X 情况，考虑 Y"（或等价短句）
- 长度 ≤20 字
- **不抄原句**
- **不用术语**（黑名单词全部剔除）
- **剔除时代局限**：只保留方法内核

## 操作步骤

### Step 2a：候选清单（列 20-30 条）

- 扫全文找"可被总结为启发式的动作/判断"
- 宁多勿少，先把网撒开
- 重复的不合并，先记下来
- 允许一条启发式对应原文多处

### Step 2b：合并去重

- 同义的归成一条
- 保留措辞最直白的那一个
- 合并后预期 10-15 条

### Step 2c：评分筛选（每条打 3 个分）

| 维度 | 含义 | 1-5 分 |
|---|---|---|
| 可操作性 | 能不能直接变成追问？变成动作？ | |
| 领域通用性 | 商业/技术/人生/学习都通用吗？还是只在某一个场景下成立？ | |
| 与 Model 契合度 | 跟这个 Model 的"定位"是否匹配？（如 Model 02 是信息校准器） | |

### Step 2d：筛出两类

1. **5 条核心启发式**（入对应 Model）
   - 三维都 ≥ 4 分
   - 与 Model 定位最契合的 5 条
   - 如果超过 5 条，选"最直接可追问"的

2. **3-5 条通用启发式**（入 `universal-heuristics.md`）
   - 跨模型通用
   - 不是这个 Model 的专属
   - 其他 Model 也能引用

## 质量检查

每条启发式问自己：
- [ ] 能用一句话说清楚吗？
- [ ] 一个没读过原文的现代人能直接用吗？
- [ ] 不含黑名单词？
- [ ] 不是原文直译？
