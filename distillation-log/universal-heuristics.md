# 通用决策启发式池

> 跨 Model 共享的启发式。Runtime 不直接输出给用户，作为分析的内部锚点。

## Model 01 贡献（4 条）

1. **具体优先**
   - 适用：决策 / 拆解 / 人际类
   - 用法：用户把一件具体事升级成"普遍困境"讨论时，把他拉回具体事——"你现在具体在跟谁、在哪件事上卡住？"

2. **事情在变**
   - 适用：任何涉及"当下判断"的分析
   - 用法：用户说"现在就这样了"时，问"半年前是这样吗？再半年呢？"——破"静态默认"

3. **两面都要看**
   - 适用：冲突 / 关系 / 评估类
   - 用法：用户只讲自己这一头时，问"对方怎么看这件事？对方现在卡在哪？"

4. **不平衡是常态**
   - 适用：任何"评估两方力量"的场景
   - 用法：用户觉得"半斤八两"时，往往是他没看清——"真的势均力敌，还是某一方其实在赢？"

## Model 02 贡献（4 条）

1. **没摸清情况先别下结论**
   - 适用：判断型问题（决策 / 评估 / 预测）
   - 用法：用户给结论前，先让他报清楚依据

2. **结论放在过程末尾，不放开头**
   - 适用：任何分析流程
   - 用法：防"结论先行，再找证据"；Model 03 / 05 / 07 都可以拿来做收口

3. **先掌握情况再定策略**
   - 适用：战略 / 战术 / 做事类问题
   - 用法：Model 05 战略战术可以借用——"你现在选打法之前，情况摸透了吗？"

4. **脑子空着出门，回来就有料**
   - 适用：卡住 / 苦思冥想的场景
   - 用法："你现在要不要先停下来问 3 个人？"——Model 07 实践论也可引用
